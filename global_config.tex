%%% ENGLISH ABSTRACT
\englishAbstract{
This project targets the formal verification of WebAssembly (WASM) programs by designing a translator from the WebAssembly Text Format (WAT) to the Boogie intermediate verification language. This translation enables the use of SMT-based verification tools such as Z3 to analyze the behavior of smart contracts compiled to WASM. The work defines formal translation rules that preserve WASM’s stack-based semantics while producing verifiable Boogie code. The project was conducted within the LIP6 laboratory in collaboration with the Fondation du Risque, with potential applications in emerging blockchain ecosystems such as XRPL.
}

\englishAbstractKeywords{Formal Verification, Smart Contracts, WebAssembly, Boogie, Translator, Blockchain, Z3}


%%% FRENCH ABSTRACT
\frenchAbstract{
Ce projet vise à permettre la vérification formelle des programmes WebAssembly (WASM) en développant un traducteur du format texte \textLR{WAT} vers le langage intermédiaire de vérification \textLR{Boogie}. Cette traduction rend possible l’analyse automatique de contrats intelligents en WASM à l’aide d’outils de vérification formelle fondés sur des solveurs SMT comme \textLR{Z3}. Le travail repose sur des règles de traduction formelles qui préservent les sémantiques de pile de WASM tout en générant du code Boogie vérifiable. Le projet a été réalisé au sein du laboratoire \textLR{LIP6}, en collaboration avec la Fondation du Risque, avec des perspectives d’application dans des blockchains modernes comme \textLR{XRPL}.
}

\frenchAbstractKeywords{Vérification formelle, Contrats intelligents, WebAssembly, Boogie, Traduction, Blockchain, Z3}
